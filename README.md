sandbox
===

Sandbox repository for idea exploration

| Name | Description |
|---|---|
| [art](art) | art projects |
| [cbtg_constraints](cbtg_constraints) | Notes on Constraint Based Tiling Generation (CBTG) constraints to add local constraints with tiles to setup problems with global or longer ranging constraints (minesweeper mine count constraint, row/column sum constraints, sudoko constraints, etc) |
| [drafts-n-dragons](drafts-n-dragons) | Experiments with "Dungeons and Diagrams" solver |
| [eznpp](eznpp) | Notes on "easy" number partition problem as a constraint satisfaction problem (abandonded, almost surely a dead-end) |
| [gilberg-figs](gilbert-figs) | Auxiliary sandbox for producing Gilbert figures for paper |
| [img](img) | Images |
| [lib](lib) | JS libraries commonly used here (some are 3rd party) |
| [molotavmarkov](molotovmarkov) | CBTG experiments for a minesweeper solver and/or maker |
| [newtmeth](newtmeth) | Auxiliary files for Newtons method fractal paper |
| [notes](notes) | Misc. notes |
| [paged-out](paged-out) | Articles for online zine [Paged Out](https://pagedout.institute/) |
| [pile](pile) | ? Lookds to be JS implementation of union find data structure? |
| [r2k2zzn](r2k2zzn) | Rectangular 2D $k=2$ Zig Zag Numberlink |
| [rng](rng) | Relative Neighborhood Graph |
| [rphp](rphp) | Rectilinear Partition Hamiltonian Path problem |
| [rprp](rprp) | Rectilinear Partition Rectilinear Polygon |
| [sfc](sfc) | Space Filling Curves |
| [sokoita](sokoita) | CBTG attempts at solving Sokoban (stalled) |
| [spinor-walk](spinor-walk) | Some experiments exploring ideas in the paper "Walks in Rotation Spaces Return Home when Doubled and Scaled" by Jean-Pierre Eckmann and Tsvi Tlustys |
| [src](src) | Misc. source files |
| [travel](travel) | Notes on places of interest for travel |
| [trochepack](trochepack) | Some experiments for 3D Gilbert curve |
| [ulhp](ulhp) | Implementation of Umans and Lenhart's Hamiltonian Cycle algorithm for 2d solid grid graphs |
| [vizlib](vizlib) | JS libraries for vizualization (common functions for two.js renderings) |
